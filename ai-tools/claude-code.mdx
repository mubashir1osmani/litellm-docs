## Quick Setup

**1. Start LiteLLM Proxy**

```yaml litellm-config.yaml
model_list:
  - model_name: claude-opus-4-5
    litellm_params:
      model: anthropic/claude-opus-4-5
      api_key: os.environ/ANTHROPIC_API_KEY
```

```bash
litellm --config litellm-config.yaml
```

**2. Point Claude Code at your proxy**

```bash
claude config set apiBaseUrl http://your-proxy:4000
```

Or set via environment:

```bash
export ANTHROPIC_BASE_URL=http://your-proxy:4000
export ANTHROPIC_API_KEY=sk-your-litellm-key
```

**3. Use Claude Code normally** — all traffic now routes through LiteLLM.
